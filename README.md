# PCB Auditor: Knight Divers
Industrial-grade PCB Netlist validation using graph topology.

## Features
- **Topology Mapping:** Traces power rails to sensitive pins.
- **Safety Checks:** Identifies short circuits and overvoltage risks.
- **Hardware Agnostic:** Designed for KiCad and Tinkercad logic.

## Task Details
- **Hard Task:** 9V-to-3.3V Voltage Mismatch identification.
- **Easy Task:** VCC-to-GND Short Circuit detection.
